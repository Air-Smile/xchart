# xunlei

迅雷远程下载服务(docker)(非官方) [链接](https://hub.docker.com/r/cnk3x/xunlei)

- 到命令行导入镜像
> docker import - cnk3x/xunlei:3.1.8 < xunlei.3.1.8.amd64.tar.xz
